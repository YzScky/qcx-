# QCX 币安合约自动交易系统

**两层指标开仓：BTC大盘环境 → 入场时机评分**

一个全自动的币安USDⓈ-M合约交易机器人。只做多，动态杠杆，硬止损保护。

## 核心策略

```
┌─────────────────────────────────────────────┐
│  🚦 第一指标：BTC大盘                        │
│  15m+1h+4h+24h四维综合判断                   │
│  ├─ 📉 向下 → 不开新仓                       │
│  ├─ ➡️ 横盘 → 正常操作                      │
│  └─ 📈 向上 → 积极操作                       │
└────────────────────┬────────────────────────┘
                     │ 通过则进入
                     ▼
┌─────────────────────────────────────────────┐
│  🎯 第二指标：入场时机（entry_timing）        │
│  扫描涨幅前20名，找评分≥10的币               │
│  按评分从高到低排序，评分越高杠杆越大         │
│  ├─ ≥16分 → 10x / 95u                        │
│  ├─ ≥13分 → 8x / 75u                         │
│  └─ ≥10分 → 5x / 55u                         │
└────────────────────┬────────────────────────┘
                     │ 开仓后进入
                     ▼
┌─────────────────────────────────────────────┐
│  💎 持仓管理（analyze_position + 硬止损）    │
│  每3分钟评估每笔持仓                         │
│  ├─ 亏损≥7% → 硬止损                        │
│  ├─ 五维评分→ ⭐止盈/ ✅持有/ ⚠️观察/ ❌止损 │
│  └─ 压力位冲击检测 → 动态调整止盈           │
└─────────────────────────────────────────────┘

  选币评分（scan_market）仅用于生成候选池排序
  → 不当作过滤条件，只看入场时机评分
```

## 功能特点

- **两层过滤**：BTC大盘环境(总开关) → 入场时机评分(精确判断)
- **自动市场扫描**：每3分钟扫描合约涨幅榜，生成候选池
- **潜力选币**：资金费率(空头挤压潜力) + 成交量爆发 + 多周期共振 + 涨幅区间
- **入场时机判断**：价格位置+15m趋势+止跌信号+量能，防止追高
- **智能止盈止损**：五维盘面评分 + 压力位冲击检测 + 动态止盈
- **硬止损保护**：亏损-7%立即平仓
- **动态仓位管理**：根据入场评分质量调整杠杆和仓位大小
- **只做多**：不做空

## 安装

```bash
# 1. 配置 Binance API
mkdir -p ~/.binance
cat > ~/.binance/trading << 'EOF'
api-key=YOUR_API_KEY
api-secret=YOUR_SECRET_KEY
env=prod
EOF

cat > ~/.binance/active_profile << 'EOF'
name=trading
EOF

# 2. 配置代理（如在中国大陆）
# SOCKS5 代理在 127.0.0.1:7891（RabbitPro/Clash）

# 3. 设置定时任务（每3分钟运行一次）
cronjob action=create \
  name="币安全自动交易系统" \
  prompt='运行 `python3 ~/.hermes/scripts/binance_strategy.py` 并输出结果' \
  schedule="every 3m" \
  repeat=9999
```

## 评分体系

### 🚦 BTC大盘（check_btc_market）
15m+1h+4h+24h四维对比涨跌数量，判断市场状态。

| 状态 | 条件 | 操作 |
|:---|:---|:---:|
| 📉 向下 | 多周期跌多涨少 | 不开新仓 |
| ➡️ 横盘 | 涨跌分化 | 正常操作 |
| 📈 向上 | 多周期涨多跌少 | 正常操作 |

### 🎯 入场时机（entry_timing）
**第二指标** — 评分≥10才开仓，评分越高杠杆越大

| 维度 | 分数 | 说明 |
|------|:----:|------|
| 价格位置 | 0~8 | 在区间中所处位置，低位高分，>70%禁止开仓 |
| 15m短期 | 0~4 | 短期涨跌方向 |
| 止跌信号 | 0~3 | 下影线≥实体2倍 |
| 缩量回调 | 0~3 | 成交量萎缩 |
| 不创新低 | 0~3 | 低点未破前低 |
| 连阴惩罚 | -5 | 连续3阴以上 |
| 放量下跌惩罚 | -5 | 量比>1.5 |

### 📡 选币排序（scan_market）
仅用来生成候选池排序，不当作过滤条件。

| 维度 | 最高分 | 说明 |
|:---|:---:|:---|
| 资金费率 | 20 | 负费率=空头挤压潜力 |
| 成交量爆发 | 15 | 近1h突然放量 |
| 多周期共振 | 15 | 15m+1h同时翻多 |
| 涨幅区间 | 10 | 8-20%启动区最高分 |

### 💎 持仓评估（analyze_position）
每3分钟评估每笔持仓，输出操作建议。

| 评分区间 | 操作 | 说明 |
|:---:|:---:|:---|
| ≥80 | ⭐ 止盈 | 趋势强+量能配合+利润达标 |
| 70~79 | ✅ 持有 | 一切正常，继续持有 |
| 50~69 | ⚠️ 观察 | 关注变化，准备减仓 |
| <50 | ❌ 止损 | 趋势走坏，立即离场 |

额外逻辑：
- 压力位冲击检测：价格接近压力位+多次冲击未破+量能衰减 → 加分止盈
- 趋势强度决定目标：强趋势8%目标，中7%，弱5%，上限15%

## 注意事项

- 需要 SOCKS5 代理（大陆访问Binance API）
- API Key 需开通合约交易权限
- 系统自动运行，无需人工干预
- 建议先用小额资金测试
- 所有API密钥存放在 `~/.binance/trading`，不提交到仓库
