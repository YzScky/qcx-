# QCX 币安合约自动交易系统

**评分选币 → 时机入场 → 动态止盈 → 硬线止损**

一个全自动的币安USDⓈ-M合约交易机器人，通过多维评分筛选高潜力币种，结合入场时机判断，实现低买高卖自动化。

## 核心策略

```
┌─────────────────────────────────────────────┐
│  ① 评分选币（scan_market）                    │
│  15m/1h/4h趋势（70%）+ 24h静态数据（30%）     │
│  → TOP30 信号                                │
└────────────────────┬────────────────────────┘
                     ▼
┌─────────────────────────────────────────────┐
│  ② 入场时机（entry_timing）                   │
│  价格位置/15m趋势/止跌/缩量/不创新低/连阴惩罚  │
│  → ≥10分才开仓                               │
└────────────────────┬────────────────────────┘
                     ▼
┌─────────────────────────────────────────────┐
│  ③ 仓位管理（place_new_trade）               │
│  评分≥20=10x/95u, ≥15=8x/75u, <15=5x/55u    │
│  → 最多6个持仓，只做多                       │
└────────────────────┬────────────────────────┘
                     ▼
┌─────────────────────────────────────────────┐
│  ④ 止盈止损（analyze_position + 硬止损）     │
│  亏损≥7%→硬止损 | 五维评分→止盈/持有/止损    │
│  → 每3分钟评估一次                           │
└─────────────────────────────────────────────┘
```

## 功能特点

- **自动市场扫描**：每3分钟扫描合约涨幅榜，筛选高潜力币种
- **多维评分系统**：K线趋势（15m/1h/4h）+ 资金费率 + 成交量 + 回调幅度
- **入场时机判断**：防止追高，等待回调到位 + 止跌信号才开仓
- **智能止盈止损**：五维盘面评分决定止盈/持有/止损
- **硬止损保护**：亏损-7%立即平仓
- **动态仓位管理**：根据评分质量调整杠杆和仓位大小
- **双向持仓模式**：支持多空双向持仓（实际只做多）

## 安装

```bash
# 1. 配置 Binance API
mkdir -p ~/.binance
cat > ~/.binance/trading << 'EOF'
api-key=YOUR_API_KEY
api-secret=YOUR_SECRET_KEY
env=prod
EOF

cat > ~/.binance/active_profile << 'EOF'
name=trading
EOF

# 2. 配置代理（如在中国大陆）
# SOCKS5 代理在 127.0.0.1:7891（RabbitPro/Clash）

# 3. 设置定时任务（每3分钟运行一次）
cronjob action=create \
  name="币安全自动交易系统" \
  prompt='运行 `python3 ~/.hermes/scripts/binance_strategy.py` 并输出结果' \
  schedule="every 3m" \
  repeat=9999
```

## 参数说明

| 参数 | 值 |
|------|-----|
| 交易方向 | 只做多 |
| 杠杆 | 动态（5x-10x） |
| 单笔仓位 | 55-95 USDT |
| 最大持仓数 | 6 |
| 止损 | -7%（硬止损） |
| 止盈 | 智能评分动态决策 |
| 运行频率 | 每3分钟 |
| 开仓评分门槛 | ≥6分（总评分）+ ≥10分（入场时机） |

## 评分体系

### 市场评分（scan_market）
| 维度 | 权重 | 说明 |
|------|------|------|
| 15m K线趋势 | 20% | 短期动能判断 |
| 1h K线趋势 | 30% | 中期趋势确认 |
| 4h K线趋势 | 20% | 大方向定调 |
| 24h静态数据 | 30% | 费率/涨幅/成交量/回调 |

### 入场时机（entry_timing）
| 维度 | 分数 | 说明 |
|------|:----:|------|
| 价格位置 | 0~8 | 低位≤25%得8分，高位>70%得0分 |
| 15m趋势 | 0~4 | 近期涨跌方向 |
| 止跌信号 | 0~3 | 下影线≥实体2倍 |
| 缩量回调 | 0~3 | 量比<0.7 |
| 不创新低 | 0~3 | 低点未破前低 |
| 连阴惩罚 | -5 | 连续3阴以上 |
| 放量下跌惩罚 | -5 | 量比>1.5 |

### 持仓评估（analyze_position）
五维分析：趋势/量价/支撑/动量/情绪 → 输出⭐止盈/持有/止损 整数评分

## 注意事项

- 需要 SOCKS5 代理（大陆访问Binance API）
- API Key 需开通合约交易权限
- 系统自动运行，无需人工干预
- 建议先用小额资金测试
- 所有API密钥存放在 `~/.binance/trading`，不提交到仓库
